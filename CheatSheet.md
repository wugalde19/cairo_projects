# Cheat Sheet

## Scarb

### New Project
```
scarb new <project-name>
```

*NOTE:* this includes git initialization, if we don't want git, we just need to include the `--no-vcs` flag

### Build and Run Project
```
scarb build
```

```
scarb cairo-run
```
